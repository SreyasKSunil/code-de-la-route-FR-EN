Code de la Route
Bilingual French English Study Web App
––––––––––––––––––––

Purpose

This project is a browser based study tool for the French Code de la Route exam.
It is designed for learners with limited French knowledge.
All content is provided in French with clear English explanations.

The goal is understanding exam logic, not memorization only.

Features

Bilingual content. French and English.
Road signs without images. - Images aren't available at the moment.
Clear rules with exam focused tips.
Driving vocabulary with examples.
Flashcards for revision.
Quiz mode with explanations.
Progress saved locally in the browser.
Works on phone and laptop.
No backend. No account required.

How to use

Open the GitHub Pages link of this repository.
Choose a module from the menu.
Study road signs, rules, and vocabulary.
Switch between French first or English first.
Use flashcards for revision.
Practice quizzes regularly.

Recommended study flow

Study one module per day.
Mark items as learned.
Review flashcards daily.
Do quiz practice every day.
Focus on mistakes and exam tips.

Content note

This project is not an official government resource.
It is not affiliated with French authorities or driving schools.
